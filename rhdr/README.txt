Cara pakai:
1. Buka file index.html di browser.
2. Tampilan awal default light mode, gunakan tombol tema untuk pindah light/dark.
3. Upload satu atau banyak file .XLS/.TXT hasil report absensi.
4. File akan disimpan ke IndexedDB browser.
5. Klik Download XLSX untuk menghasilkan 1 workbook dengan banyak sheet.
6. Klik Hapus Data Lokal untuk mengosongkan IndexedDB.
Catatan:
- Aplikasi ini membaca file .XLS yang sebenarnya berisi teks report, seperti contoh file lampiran.
- Pilihan tema tersimpan otomatis di browser.
- Sheet Excel tetap menggunakan tampilan gelap dengan blok absensi berwarna.
